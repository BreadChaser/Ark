# Freedesktop sound samples

The MP3 files in this directory are normalized, format-converted copies from
`sound-theme-freedesktop` 0.8. Ark's original WAV sounds are not part of that
theme.

| Ark file | Upstream file |
| --- | --- |
| `done-arrival.mp3` | `service-login.oga` |
| `done-spark.mp3` | `message-new-instant.oga` |
| `done-resolve.mp3` | `trash-empty.oga` |
| `input-warning.mp3` | `dialog-warning.oga` |
| `input-urgent.mp3` | `suspend-error.oga` |
| `input-busy.mp3` | `phone-outgoing-busy.oga` |

Upstream source: <https://www.freedesktop.org/wiki/Specifications/sound-theme-spec/>
