Manual follow-up

The laptop has a world-writable system SSH include. Fix it locally with:

sudo chmod 644 /etc/ssh/ssh_config.d/20-systemd-ssh-proxy.conf

This does not block ARK's VM-to-laptop route, but it prevents locally running Codex from trusting the laptop's SSH client configuration.
